===========
photovoltaic
===========

photovoltaic provides basic function used in photovoltaics. Typical usage:

    import photovoltaic as pv

    irradiance = pv.blackbody_spectrum(6000)
    print(irradiance)

This would print the blackbody irradiance in W/m2/nm
